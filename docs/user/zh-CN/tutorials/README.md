# 教程

[English](../../en/tutorials/README.md) | 简体中文

所有[入门页](../README.md)殊途同归,终点都是 `vesicle doctor` 通过、并在项目目录里打开 Vesicle。从这里开始学怎么真正用它创作。

教程假设你已经能启动 Vesicle、`vesicle doctor` 报 `Missing: none`。每篇只引入当下需要的概念,结尾给出检查点。

## 学习路径

1. [第一次对话:界面与门](./first-conversation.md) —— 认识界面,理解"门"这种暂停确认的交互。
2. [运行中继续工作](./work-while-running.md) —— 用队列补充主任务,用 `/btw` 做不进入主对话的旁路提问。
3. [第一张角色卡](./first-character-card.md) —— 走完 ETL 工作流 A,产出一张通过校验的 Module A 角色卡。
4. [第一张情景卡](./first-scenario-card.md) —— 基于角色卡产出 Module B 情景卡并校验。
5. [会话恢复与回退](./sessions-and-rewind.md) —— 恢复历史会话,回退到任意一步。
6. [为项目建立持久化指令](./persistent-instructions.md) —— 用 `/init`、`/instructions` 和模型工具维护跨会话项目规范。
7. [权限与宿主 Shell](./permissions-and-shell.md) —— 四档工具批准模式,以及何时启用 shell。

做完这七篇,你就掌握了 Prism Vesicle 的日常创作循环。命令速查、配置、安全模型等参考内容见[参考区](../reference/README.md)。
